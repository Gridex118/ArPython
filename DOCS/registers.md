# PVM registers

## General Purpose Registers
- Rax (32 bit)
- Rbx (32 bit)
- Rcx (32 bit)
- Rdx (32 bit)
- Rah (16 bit)
- Ral (16 bit)
- Rbh (16 bit)
- Rbl (16 bit)
- Rch (16 bit)
- Rcl (16 bit)
- Rdh (16 bit)
- Rdl (16 bit)

## Special Registers
- Stack Pointer: Rsp (16 bit)
- Program Counter: Rpc (16 bit)
